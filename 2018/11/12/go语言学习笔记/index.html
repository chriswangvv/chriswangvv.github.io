go语言学习笔记

  • 2018-11-12

基本

  1. 源文件非注释第一行指明属于哪个包；每个go应用都包含一个名为main的包
  2. import “包名”;fmt 包实现了格式化 IO（输入/输出）的函数
  3. 函数执行顺序：有init()函数则先执行init函数，没有则启动主函数（main()）
  4. 当标识符（包括常量、变量、类型、函数名、结构字段等等）以一个大写字母开头时，使用这种形式的标识符的对象能被外部包的代码所使用（客户端程序需要先导入这个包），这被称为导出（相当于public）；标识符以小写字母开头，则只对包内可见可用（相当于protected）

注意：

  1. 左大括号不能单独放在一行

  2. 一行代表一个语句结束，不需要分号（如果多个语句写在一行则需要分号：不要这么写）

  3. 第一个字符必须是字母或下划线

数据类型

  1. 布尔型
  2. 数字型: 整型：int; 浮点型：float32 float64；（还有一些别的，我感觉暂时用不上）

  4. 派生类型：
    1. 指针类型（Pointer）

    3. 结构化类型
    4. Channel类型

变量

声明变量

var

  1. 单个：
    1. var name type
    2. var name = value
    3. name := value;(name不能被声明过、只能写在函数体内)
  2. 多个同上（个数必须得对应）（并行/同时 赋值）
值类型和引用类型
  1. 所有基本类型（int float bool string）属于值类型
  2. &i 来获取变量 i 的内存地址

注意：

  1. 这种因式分解关键字的写法一般用于声明全局变量
    `
    var (
    vname1 v_type1
vname2 v_type2
    )`
  2. 声明的局部变量必须使用
  3. 全局变量允许声明但不使用
  4. 想要交换两个变量的值，则可以简单地使用 a, b = b, a，两个变量的类型必须是相同。
  5. 空白标识符 _ 也被用于抛弃值，如值 5 在：_, b = 5, 7 中被抛弃。

  6. _ 实际上是一个只写变量，你不能得到它的值。这样做是因为 Go 语言中你必须使用所有被声明的变量，但有时你并不需要使用从一个函数得到的所有返回值。（这条没细看）

  7. 并行赋值也被用于当一个函数返回多个返回值时，比如这里的 val 和错误 err 是通过调用 Func1 函数同时得到：val, err = Func1(var1)。（这条没细看）

常量

常量基本用法
  1. 声明常量
    1. 显式类型定义： const b string = “abc”
    2. 隐式类型定义： const b = “abc”

  2. 可以用作枚举

    const (
    Unknown = 0
    Female = 1
    Male = 2    
)

  3. 常量可以用len(), cap(), unsafe.Sizeof()函数计算表达式的值。常量表达式中函数必须是内置函数（搜go 内置函数）

    const (
    a = "abc"
    b = len(a)
    c = unsafe.Sizeof(a)
)
func main() {
    printlin(a, b, c)//abc 3 16
}

    字符串类型在 go 里是个结构, 包含指向底层数组的指针和长度,这两部分每部分都是 8 个字节，所以所有字符串类型大小为 16 个字节。

  4. 如果同时声明多个常量，只赋值了第一个，那么下面的值会跟第一个一样

    const (
    x          // 报错，没有赋值
    z = "yyy"  // yyy
    y          // yyy
)
    iota

    iota 特殊常量，一个可以被编译器修改的常量

    iota 在 const关键字出现时将被重置为 0，const 中每新增一行常量声明将使 iota 计数一次(iota 可理解为 const 语句块中的行索引)。

    const (
    a = iota   //0
    b  //1
    c  //2
    d = "ha"   //独立值，iota += 1
    e  //"ha"   iota += 1
    f = 100//iota +=1
    g  //100  iota +=1
    h = iota   //7,恢复计数
    i  //8
)

注意：
此处例子涉及左移右移

const (
    i=1<<iota   // i左移0位，不变
    j=3<<iota   // 左移一位，变为二进制110，即6
    k //左移 2 位,变为二进制 1100, 即 12;
    l // 左移 3 位,变为二进制 11000,即 24
)
