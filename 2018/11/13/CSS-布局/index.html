display属性

  1. block
    <div>是一个标准的块级元素。一个块级元素会新开始一行并尽可能撑满容器。其他常用的块级元素包括<p>、<form>和 HTML5中的新元素<header> <footer> <section>等待
  2. inline
    <span>是一个标准的行内元素。a 元素是最常用的行内元素。

  3. none
    一些特殊元素的默认 display 值是它，例如 <script> 。 display:none 通常被 JavaScript 用来在不删除元素的情况下隐藏或显示元素。

    它和 visibility 属性不一样。把 display 设置成 none 元素不会占据它本来应该显示的空间，但是设置成 visibility: hidden; 还会占据空间。

  4. 其他 display 值
    list-item 和 table inline-block 和 flex

例子

margin: auto;

设置块级元素的 width 可以防止它从左到右撑满整个容器。然后你就可以设置左右外边距为 auto 来使其水平居中。元素会占据你所指定的宽度，然后剩余的宽度会一分为二成为左右外边距。

唯一的问题是，当浏览器窗口比元素的宽度还要窄时，浏览器会显示一个水平滚动条来容纳页面

<div id="main">aaa</div>

#main {
    width: 600px;
    margin: 0 auto;
    background-color: #eeeeee;
}
max-width

在这种情况下使用 max-width 替代 width 可以使浏览器更好地处理小窗口的情况。这点在移动设备上显得尤为重要

#main {
    max-width: 600px;
    margin: 0 auto;
    background-color: #eeeeee;
}

box-sizing: border-box;

两个div都加了box-sizing: border-box;属性后，宽被固定成了500px,边框的宽度和padding都往里了。

position

  1. static 默认值。任意 position: static; 的元素不会被特殊的定位

  2. relative 相对定位
    relative 表现的和 static 一样，除非你添加了一些额外的属性

    在一个相对定位的元素上设置 top 、 right 、 bottom 和 left 属性会使其偏离其正常位置。其他的元素的位置则不会受该元素的影响发生位置改变来弥补它偏离后剩下的空隙。

  3. fixed 固定定位
    会相对视窗来定位，即使页面滚动，他还是会停留在相同的位置。 top 、 right 、 bottom 和 left 属性都可用。

    一个固定定位元素不会保留它原本在页面应有的空隙（脱离文档流）

  4. absolute
    与fixed表现类似，但不是相对于视窗 而是相对于最近的有position属性（非static）的祖先元素。如果绝对定位的元素没有非static的祖先元素，那么它是相对于文档的 body 元素，并且它会随着页面滚动而移动。
