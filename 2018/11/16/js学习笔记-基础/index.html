  1. eval() 避免用吧

    执行指定代码之后的返回值。如果返回值为空，返回undefined

  2. isFinite()

> isFinite(testValue) 函数用来判断被传入的参数值是否为一个有限数值（finite number）

  1. isNaN()

    isNaN(testValue) 函数用来确定一个值是否为NaN
    也可以用ES5/ES6中定义的Number.isNaN()或typeof()来判断是否为一个非数字

  2. parseFloat()

    parseFloat(value) 函数解析一个字符串参数并返回一 个浮点数

    如果给定值不能被转换成数值，则会返回 NaN。

  3. parseInt()

    parseInt(string, radix) 函数解析一个字符串参数，并返回一个指定基数的整数 (数学系统的基础)

    如果被解析参数的第一个字符无法被转化成数值类型，则返回 NaN

    注意：radix参数为n 将会把第一个参数看作是一个数的n进制表示，而返回的值则是十进制的。例如

    parseInt('123', 5) // 将'123'看作5进制数，返回十进制数38 =1*5^2 + 2*5^1 + 3*5^0 = 38

  1. decodeURI()

    decodeURI(encodedURI) 函数解码一个由encodeURI 先前创建的统一资源标识符（URI）或类似的例程

  2. decodeURIComponent()

    decodeURIComponent(encodedURI) 方法用于解码由 encodeURIComponent 方法或者其它类似方法编码的部分统一资源标识符（URI）

  3. encodeURI()

    encodeURI(URI) 函数通过将特定字符的每个实例替换为一个、两个、三或四转义序列来对统一资源标识符 (URI) 进行编码 (该字符的 UTF-8 编码仅为四转义序列)由两个 “代理” 字符组成)。

    返回 一个新字符串, 表示提供的字符串编码为统一资源标识符 (URI)。

  4. encodeURIComponent()

  1. charAt() 方法可返回指定位置的字符

  2. 连等

    判断3个数连等为什么不能用a ==b ==c

    由于==是从左到右进行运算的,对于表达式 ( a == b == c ),首先会进行 ( a == b )的逻辑运算,然后得出逻辑值1或0,得到的结果作为返回值进行 ( 返回值 == c ) 的逻辑运算,然后得出逻辑值1或0,作为整个表达式的返回值.
    所以也就是判断 ( ( a == b ) == c )即 ( 1或0 == c ),因此不能这样用.
