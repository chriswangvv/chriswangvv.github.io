持续更新-关于CSS，一些容易忘记的事

发表于 2018-11-16 | 分类于 CSS

基本

  1. 选择器权重
    1. 内联样式，如 style = "",权值为1000
    2. ID选择器，如 #content,权值为100
    3. 类、伪类和属性选择器，如 .content,权值为10
    4. 类型选择器和伪元素选择器，如 div p,权值为1
  2. 优先级
    1. 权值相等时，后出现的样式覆盖先出现的样式表设置
    2. 创作者的规则高于浏览者
    3. 继承的 CSS 样式不如后来指定的 CSS 样式
    4. 在同一组属性设置中标有!important规则的优先级最大

动画

  1. @keyframes规则内指定一个 CSS样式和动画将逐步从目前的样式更改为新的样式。

边框

  • border-radius
  • box-shadow
  • border-images

背景

  • background-size
    • 以像素或百分比规定尺寸。如果以百分比规定尺寸，那么尺寸相对于父元素的宽度和高度
  • background-origin

字体

@font-face

@font-face {
font-family: 'FontAwesome';
src: url('fonts/fontawesome-webfont.woff');
}

多列

  • column-count: 指定元素的列数
  • column-rule： 列之间的宽度 样式 颜色
  • column-gap： 列之间的差距

UI

  • resize
  • box-sizing
  • outline-offset

过渡效果

  • transition-property // 应用过渡的css属性的名称
  • transition-duration // 过渡效果花费的时间
  • transition-timing-function // 规定过渡效果的时间曲线。默认是 “ease”
  • transition-delay

文本效果

  • text-shadow
  • word-wrap

2D转换

  • translate()
  • rotate()
  • scale()
  • skew()
  • matrix()

3D转换

  • rotateX()
  • rotateY()
  • rotateZ()
  • rotate3d(x,y,z,a)

text-indent: 文本块中首行文本的缩进

direction: 规定文本的方向 / 书写方向
  1. 可取值： ltr rtl
  2. 默认值： ltr（从左到右）
  3. 可继承
  4. object.style.direction=”rtl”
  5. 设置父div的rtl属性，子元素显示出来

  6. 改变的只是内联元素块的左右顺序

    内联元素块：

    <img> <button> <input> <video> <object>

或者inline-block水平的元素

    text-align: start; end;跟margin-block-*和margin-inline-*似的

  1. transform:rotate(7deg);

  2. animation

    animation: dot 3s infinite step-start both

@keyframes dot {
    33% {
        transform: translateY(-2em);
    }
    66% {
        transform: translateY(-1em)
    }
}

  3. white-space 属性设置如何处理元素内的空白

    normal: 空白被浏览器忽略（默认值）

    pre：空白被浏览器保留。类似<pre></pre>

    nowrap: 文本不换行，直到遇见<br>标签

    pre-wrap：保留空白符序列，正常换行

    pre-line：合并空白符序列，保留换行符

    inherit：从父元素继承white-space的值

  4. ::after,::before

    这些不会出现在DOM中，不会改变文档内容，不可复制，仅仅在css渲染层加入。不要用它展示有实际意义的内容，尽量显示修饰性内容

    1. 例子

      .number::befor{content: '\260E'}
<div class="number"></div>

    2. content:content必须有值，至少是空；伪类元素默认为inline

      1. string:

      2. attr():调用当前元素的属性

        content: attr(href)

      3. url() / uri():引用媒体文件，图片啥的

      4. counter()：计数器，配合counter-increment和counter-reset

        body {
    counter-reset: section;
}

h1 {
    counter-reset: subsection;
}

h1::before {
    counter-increment: section;
    content: counter(section) "、";
}

h2::before {
    counter-increment: subsection;
    content: counter(section) "." counter(subsection) "、"
}

<h1>test h1</h1>
<h2>test h2</h2>
<h3>test h3</h3>

      5. blockquote的 open-quote：设置Content是开口引号

    3. 使用

      1. 清浮动

        .cf:before, .cf:after {
    content: " ";
    display: table;
}
.cf:after {
    clear: both;
}
.cf {
    *zoom: 1;
}

      2. 模拟float:center

      3. 图形效果 小图标

      4. 显示打印网页的url

        @media print {
  a[href]:after {
    content: " (" attr(href) ") ";
  }
}

  5. 裁剪图片

    clip: rect(0px,60px,60px,10px);

  6. css 单位 px em rem

    em：相对单位，基准点为父节点字体的大小

    浏览器默认1em=16px，所以0.75em=12px，这样不好算，所以在根元素上写font-size: 62.5%,这样1em = 10px,12px=1.2em

    em会根据父级元素的大小而变化，rem相对于根基元素不被父类影响

    rem chrome/firefox/IE9+支持
