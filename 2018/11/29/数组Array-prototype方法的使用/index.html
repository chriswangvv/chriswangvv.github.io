数组Array.prototype方法的使用

在ES5中，一共有九个Array方法:

  • Array.prototype.indexOf
  • Array.prototype.lastIndexOf
  • Array.prototype.every
  • Array.prototype.some
  • Array.prototype.forEach
  • Array.prototype.map
  • Array.prototype.filter
  • Array.prototype.reduce
  • Array.prototype.reduceRight

indexOf()

返回在该数组中第一个找到的元素的位置，如果不存在则返回-1

arr.indexOf('wrr')

lastIndexOf()

返回在该数组中最后一个找到的元素的位置，如果不存在则返回-1
（可以用这两个判断是否有重复元素吧）

every()

检测数组中每一项是否符合条件

let arr1 = [1, 2, 51, 98]
let result = arr1.every(function(item，index) { // index: 索引; item： 值
    return item > 0
})

some()

检测数组中是否有某一项符合条件

var result0 = arr.some(function(item, index) {
    return item < 0
})

相对比于用for循环来写，确实用every和some代码更简介语法更明确

forEach()

为每个元素执行对应的方法

arr.forEach(function(item, index) {
    console.log(index, item)
})

用来替换for循环

map()

对数组的每个元素进行一定操作（映射）后，会返回一个新的数组

map()是处理服务器返回数据时是一个非常实用的函数。

用for循环：

function getNewArr() {
    var newArr = [];
    for (var i = 0; i < oldArr.length; i++) {
        var item = oldArr[i];
        item.full_name = [item.first_name, item.last_name].join(" ")
        newArr[i] = item
    }
    return newArr
}

不用for循环：

function getNewArr() {
    return oldArr.map(function(item, index) {
        item.full_name = [item.first_name, item.last_name].join(" ")
        return item
    })
}

filter()

创建一个新的匹配过滤条件的数组

const arr0 = arr.filter(function(item, index) {
    return item.name === "orange"
})

reduce()**

可以实现一个累加器的功能，将数组的每个值（从左到右）将其降低到一个值

实例： 输出数组中元素出现了几次

for循环：

function getWordCnt() {
    var obj = []
    for (var i = 0; i < arr.length; i++) {
        var item = arr[i]
        obj[item] = (obj[item] + 1) || 1
    }
    return obj
}

reduce：

function passValue() {
    return arr.reduce(function(prev, next) {
        prev[next] = prev[next] + 1 || 1
        return prev
    }, {})
}

reduceRight()

reduceRight的语法以及回调函数的规则和reduce方法是一样的，区别就是在与reduce是升序，即角标从0开始，而reduceRight是降序，即角标从arr.length-1开始

方法可应用于字符串

实例：字符串倒序

window.onload = function () {
    var result = [].reduceRight.call(word, appendToArray, "the ")
    console.log(result) // the computer
}

var word = "retupmoc"

function appendToArray(previousValue, currentVAlue){
    return previousValue + currentValue
}

isArray()

Array对象的一个静态函数，用来判断一个对象是不是数组

Array.isArray(arr)

备注

检测数组中每一项是否符合条件 和 检测数组中是否有某一项符合条件

用for循环来做：

  1. 检测数组中每一项是否符合条件：是否全部大于0
var ary = [12,23,24,42,1];
var result = function(){
    for (var i = 0; i < ary.length; i++) {
        if(ary[i] < 0){
           return false;
        }
      }
      return true; //需全部满足
}
  1. 检测数组中是否有某一项符合条件：是否存在小于0的项
var ary = [12,23,-24,42,1];
var result = function(){
    for (var i = 0; i < ary.length; i++) {
        if(ary[i] < 0){
               return true;
        }
      }
      return false; //只需满足一个
}

forEach()和map()的区别

  • 语法：forEach和map都支持2个参数：一个是回调函数（item,index,list）和上下文；

    1. forEach：

      1. 用来遍历数组中的每一项；

2. 这个方法执行是没有返回值的，对原来数组也没有影响；

3. 数组中有几项，那么传递进去的匿名回调函数就需要执行几次；

4. 每一次执行匿名函数的时候，还给其传递了三个参数值：数组中的当前项item,当前项的索引index,原始数组list；

5. 理论上这个方法是没有返回值的，仅仅是遍历数组中的每一项，不对原来数组进行修改；但是我们可以自己通过数组的索引来修改原来的数组；

6. forEach方法中的this是ary,匿名回调函数中的this默认是window

     var arr = [12, 23, 24, 42, 1];
     var res = arr.forEach(function(item, index, input) {
         input[index] = item * 10
     })
     console.log(res) // undefined
     console.log(arr) // [120, 230, 240, 420, 10]

  • 区别

    1. map:

      1. map的回调函数中支持return返回值；return的是啥，相当于把数组中的这一项变为啥（**并不影响原来的数组**，只是相当于把原数组克隆一份，把克隆的这一份的数组中的对应项改变了）；

      var res = arr.map(function(item, index, input) {
          return item * 10
      })
      console.log(res) // [120, 230, 240, 420, 10]
      console.log(arr) // [12, 23, 24, 42, 1]
