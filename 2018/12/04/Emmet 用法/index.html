Emmet 用法

发表于 2018-12-04 | 分类于 其他（相关）

整理自官方文档

缩写是Emmet工具包的核心

我们写别的代码的时候，为了提高可读性（之类的）需要在元素与元素之间加空格，但是在emmet中空格是停止符号，所以不必加

缩略语

例子：

#page>div.logo+ul#navigation>li*5>a{Item $}

生成：

<div id="page">
    <div class="logo"></div>
    <ul id="navigation">
        <li><a href="">Item 1</a></li>
        <li><a href="">Item 2</a></li>
        <li><a href="">Item 3</a></li>
        <li><a href="">Item 4</a></li>
        <li><a href="">Item 5</a></li>
    </ul>
</div>

语法

元素

通过使用div元素名称生成div元素

嵌套运营商

  1. 子元素 >

    div>ul>li

  2. 兄弟元素（Sibling） +

    div+p+bq

> bq: `<blockquete></blockquete>`

  1. 爬上 (往上升一层，多个^升多层) ^

    例子1：

    div+div>p>span+em^bq

    生成：

    <div></div>
<div>
    <p><span></span><em></em></p>
    <blockquote></blockquote>
</div>    
// blockquote元素将与p元素为兄弟元素    

例子2：

    div+div>p>span+em^^^bq

生成：

    <div></div>
    <div>
        <p><span></span><em></em></p>
    </div>
    <blockquote></blockquote>

  1. 生成多个一样的 *

    例子:

    ul>li*5

    生成:

    <ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
</ul>

  2. 分组 ()

    例子1：

    div>(header>ul>li*2>a)+footer>p

    生成：

    <div>
    <header>
        <ul>
            <li><a href=""></a></li>
            <li><a href=""></a></li>
        </ul>
    </header>
    <footer>
        <p></p>
    </footer>
</div>

    例子2：

    (div>dl>(dt+dd)*3)+footer>p

    生成：

    <div>
    <dl>
        <dt></dt>
        <dd></dd>
        <dt></dt>
        <dd></dd>
        <dt></dt>
        <dd></dd>
    </dl>
</div>
<footer>
       <p></p>
</footer>

属性运算符

  1. id和class

    语法： div#id div.class div#id.class.class

例子：

    div#header+div.page+div#footer.class1.class2.class3

生成：

    <div id="header"></div>
    <div class="page"></div>
    <div id="footer" class="class1 class2 class3"></div>

  1. 自定义属性

    语法： 元素名[自定义属性=值 自定义属性=值]

    例子：

    a[routerLink="/app/route"]

    生成：

    <a routerLink="/app/route"></a>

    例子2：

    [a='value1' b='value2']

    生成2：

    <div a="value1" b="value2"></div>

  1. 编号（第一个例子中的li里的a标签）: $

    例子：

    ul>li.item$*5

    生成：

    <ul>
    <li class="item1"></li>
    <li class="item2"></li>
    <li class="item3"></li>
    <li class="item4"></li>
    <li class="item5"></li>
</ul>

    1. 更改编号基数和方向

      例子：

      ul>li.item$@-3*5

      生成：

      <ul>
    <li class="item7"></li>
    <li class="item6"></li>
    <li class="item5"></li>
    <li class="item4"></li>
    <li class="item3"></li>
</ul>

      例子2：

      ul>li.item$@-*5

      生成2：

      <ul>
    <li class="item5"></li>
    <li class="item4"></li>
    <li class="item3"></li>
    <li class="item2"></li>
    <li class="item1"></li>
</ul>

内容 {}

例子： 

a{Click me}

生成：

<a href="">Click me</a>

a{click}和a>{click}、b{here}和b>{here}分别输出相同的, 但是 a{click}+b{here} 和 a>{click}+b{here}输出不同

a{click}+b{here} ：

<a href="">click</a><b>here</b>

a>{click}+b{here}：

<a href="">click<b>here</b></a>

p>{Click }+a{here}+{ to continue}:

<p>Click <a href="">here</a> to continue</p>

p{Click }+a{here}+{ to continue}:

<p>Click </p>
<a href="">here</a> to continue

隐式标签名称

例子：

.class

生成：

<div class="class"></div>

例子1：

table>.row>.col

生成1：

<table>
    <tr class="row">
        <td class="col"></td>
    </tr>
</table>

例子2：

em>.class

生成2：

<em><span class="class"></span></em>

例子3： 

ul>.class

生成3：

<ul>
    <li class="class"></li>
</ul>
