一些less的基本用法

发表于 2018-12-10 | 分类于 CSS

注释

less中的 /***/ 注释，注释中的编译后会被保留

// 注释，注释中的编译后不保留

变量

写法： @变量名：变量值

例子： 

@test_width: 300px;
.box {
    width: @test_width;
    height: @test_width;
}

混合

  1. 混合

    例子：

 .box{
     .border;
 }

 .border {
     border: 1px solid #000000
 }

  2. 混合 可带参数 传值时带单位

    .box {
    .border1(5px);
}

.border1(@border_width) {
    border: @border_width solid red
}

  3. 混合 有默认值

    .border3(@border_width: 10px) {
    border: @border_width solid green
}

匹配模式

  1. 匹配模式

    例子：

    .triangle(top, @w: 5px, @c: #ccc) {
    border-width: @w;
    border-color: transparent transparent @c transparent;
    border-style: dashed  dashed solid dashed
}
.triangle(bottom, @w: 5px, @c: #ccc) {
    border-width: @w;
    border-color: @c transparent transparent  transparent;
    border-style: solid dashed  dashed  dashed
}
.triangle(left, @w: 5px, @c: #ccc) {
    border-width: @w;
    border-color: transparent @c transparent transparent;
    border-style: dashed solid dashed  dashed
}
.triangle(right, @w: 5px, @c: #ccc) {
    border-width: @w;
    border-color:  transparent transparent   transparent @c;
    border-style:  dashed  dashed  dashed solid
}

.triangle(@_, @w: 5px, @c: #ccc) {
    // 不管@_变量值是多少都会带着这一些属性
    width: 0px;
    height: 0px;
    overflow: hidden;
    margin-bottom: 20px;
}
.top {
    .triangle(top)
}
.bottom {
.triangle(bottom)
}

  2. 匹配模式 定位 例子

    例子：

 .pos(a) {
     position: absolute;
 }
 .pos(r) {
     position: relative;
 }
 .pos(f) {
     position: fixed;
 }

 .pipis {
     width: 200px;
     height: 200px;
     .pos(r)
 }

运算

例子：

@test_01: 300px;
.test {
    width: @test_01 + 20;
    color: #ccc - 10; // 颜色的用得不多
}

嵌套规则

嵌套层数不要太多，最好不要超过三层

a {
    color: #000000;
    text-decoration: none;
    &:hover { // & 代表上一层选择器，这个上一层代表的是css中写的上一层
        color: blue
    }
}

例子2：

.title{
    &_nav{}
}

arguments

用得不多

例子：

.border_arg (@w: 1px; @c: #ccc; @xx: solid) {
    border: @arguments
}

避免编译

  1. 避免编译

    ~''

    width: ~'calc(300px - 30px)';

  2. !important

其他

  1. 清除浮动

    .clearfix:after {

    content: "";
display: block;
clear: both

    }
    .clearfix {

    zoom: 1;

    }

    .clearfix {

    &:after {
    content: "";
    display: block;
    clear: both
}
zoom: 1;

    }

  2. 导入

    @import “ku” // less
    @import (less) “a.css” // css import写在哪就会出现在哪
