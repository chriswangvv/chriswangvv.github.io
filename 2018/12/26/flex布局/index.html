flex布局

发表于 2018-12-26 | 分类于 CSS

不建议对flex子项使用width属性，因为不够弹性；推荐用flex-basis

都是控制flex子项的呈现

作用在flex容器上

控制整体

  • flex-direction

    控制子项整体布局方向

    1. row: 默认值。方向为文档水平流方向（direction：rtl）
    2. row-reverse：与row相反的行
    3. column： 列
    4. column-reverse： 与column相反的列

  • flex-wrap

    控制子项整体单行显示还是换行显示

    1. nowrap

      单行显示，不换行。容易宽度溢出。

    2. wrap： 宽度不足时换行

    3. wrap-reverse： 宽度不足时换行，从下往上

  • flex-flow

    flex-direction和flex-wrap的缩写

    例如： flex-flow: row-reverse wrap-reverse;

  • justify-content

    水平方向子项的对齐和分布方式

    1. flex-start: 默认左对齐
    2. flex-end：默认右对齐
    3. center： 居中对齐，空白元素在两边

    4. space-between： 两端对齐，空白元素

    5. space-around： 两边的空白是中间空白的一半

    6. space-evenly： 每个flex子项两侧空白间距相等

  • align-items

    items指的是flex的子项们。flex的子项相对于flex容器在垂直方向上的对齐方式

    1. stretch 默认值。如果flex子项没设置高度或设置成auto，flex子项拉伸到容器高度；如果设置了高度，按高度值渲染
    2. flex-start 默认为容器顶部对齐
    3. flex-end 默认为容器底部对齐
    4. center 垂直居中对齐
    5. baseline 相对于flex容器第一行文字的基线对齐 **

  • align-content

    与justify-content。指明垂直方向每一行flex元素的对齐和分布方式。如果只有一行，那么不起作用。（还不能是flex-direction是column的）

    1. stretch 默认值。不设置子元素高度时，每一行flex子元素等比例拉伸

    2. flex-start 顶部堆砌

    3. flex-end 底部堆放

    4. center 垂直居中对齐

    5. space-between 上下两行两端到头对齐，剩下每一行元素等分剩余空间。

    6. space-around 两头的空白是中间空白的一半

    7. space-evenly 每一行元素都完全上下等分

作用在flex子项上

控制个体

  • order 改变某一个子项的排序位置

    默认值为0，数字越小越靠前

  • flex-grow 分配剩余空白间隙拓展自己

    1. 如果只有一个flex子项设置了flex-grow属性值：

      • 如果flex-grow值小于1，则扩展的空间就总剩余空间和这个比例的计算值
      • 如果flex-grow值大于1，独享所有剩余空间

    2. 如果有多个flex子项设置了flex-grow属性值：

      • 如果flex-grow值总和小于1，则按比例扩展，留空白
      • 如果flex-grow值总和大于1，按比例扩展，不留空白。例如所有flex子项都是flex-grow：1，那么等于空白大家等分

  • flex-shrink 处理flex容器空间不足时，单个元素的收缩比例

    1. 当flex的容器空间不足的时候，默认为1每个子项等比例收缩(即设子项宽度为80但实际可能只有60)
    2. 当设置单个子项的flex-shrink值时，这个子项按设置的收缩，另外的按剩下的容器等比例收缩（只有这个子项值特别大的时候剩下的才有可能不收缩，我试的是两万）。这个子项的flex-shrink值为0时，这个子项不收缩，剩下的等比例收缩
    3. 当每个子项都设置值的时候，按值的比例收缩；每个子项值都为0时，溢出flex容器

  • flex-basis 定义了在分配剩余空间之前元素的默认大小

    1. 默认值auto；设置了width就占据width，没有就按内容宽度来
    2. 如果同时设置width和flex-basis，就渲染表现看会忽略width
    3. 剩余空间不足时会收缩，如flex-shrink所言

  • flex 是 flex-grow flex-shrink flex-basis的缩写，后两个参数可选，默认值为0 1 auto

    1. flex： none => flex： 0 0 auto
    2. flex： auto => flex： 1 1 auto

  • align-self 控制单独某一个flex子项的垂直对齐方式（跟写在flex容器上的align-items属性一致）

    1. align-self: auto | flex-start | flex-end | center | baseline | stretch;
    2. auto 默认值，继承flex容器的align-items属性值
